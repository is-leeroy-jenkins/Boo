# Boo Documentation

Boo is a multi-provider AI assistant application that integrates Streamlit, OpenAI, Google Gemini,
xAI Grok, document workflows, image workflows, audio workflows, vector storage, and local data
utilities.

## Purpose

This documentation provides operational, configuration, architectural, and API-level references for
the Boo application.

The documentation is generated from:

- project Markdown files in the `docs/` directory;
- Google-style Python docstrings in the source modules;
- provider wrapper classes for GPT, Gemini, and Grok;
- Streamlit application helpers and workflow functions.

## Main Sections

- **Getting Started** explains installation and local execution.
- **Configuration** explains environment variables and provider keys.
- **User Guide** explains application workflows.
- **Architecture** explains provider routing and application structure.
- **API Reference** is generated from source-code docstrings.
